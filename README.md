# Google Play Store Apps - Comprehensive EDA

## Overview
This repository contains a comprehensive Exploratory Data Analysis (EDA) of the Google Play Store Apps dataset. The analysis focuses on data cleaning, transformation, and deriving meaningful business insights from app-related attributes such as category, ratings, installs, and reviews.

The project demonstrates:
- Stepwise data preprocessing and cleaning
- Exploratory analysis with visualizations
- Business-driven insights for app performance
- Structured workflow for reproducibility

- Data cleaning and handling of missing values
- Conversion of categorical and numerical features
